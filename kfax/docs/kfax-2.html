2. Installation

2.1 How to obtain KFax

KFax is part of the KDE project http://www.kde.org. KFax can be found on ftp://ftp.kde.org/pub/kde/, the main ftp site of the KDE project.

2.2 Requirements

In order to successfully compile KFax, you need the latest versions of libkdecore, libkfm and libkdeui. All required libraries as well as KFax itself can be found on ftp://ftp.kde.org/pub/kde/.

2.3 Compilation and installation

In order to compile and install KFax on your system, execute the following commands in the base directory of the KFax distribution: 

% ./configure
% make
% make install

Since KFax uses autoconf you should have not trouble compiling it. Should you run into problems please report them to the KDE mailing lists.
