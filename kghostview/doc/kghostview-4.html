4. Primary menu items

4.1 File

New

Creates a new KGhostview window.

Open ...

Lets you select a file to open.

Close

Closes the current KGhostview window.

Print ...

Lets you print the document previewed in this KGhostview window.

Entries 1. - 4.

The last four opened documents are stored in these menu entries. Simply select one of these to reopen that document.

Quit

Exits KGhostview, closing all the windows.

4.2 View

Zoom in

Increases the magnification of the view of your document.

Zoom out

Decreases the magnification of the view of your document.

View control ...

Lets you set the magnification, orientation and page size for the view of your document.

Shrink wrap

Resizes the KGhostview window to neatly fit the width of the document. If the document is too wide to fit on the screen then the KGhostview window is stretched to the maximum possible width.

Redisplay

Renders the current page of your document again.

Info ...

Displays some basic information about your document.

4.3 Go

Next page

Moves one page forward in the document.

Previous page

Moves one page back in the document.

Go to page ...

Lets you choose which page to display.

Go to start

Goes to the start of the document.

Go to end

Goes to the end of the document.

Read down

Scrolls up the current page for convenient reading of the document in chunks.

4.4 Options

Turn antialias off (on)

Antialiasing is a rendering technique that improves the appearance of material to be viewed on low resolution devices like computer monitors. Only recent versions of Ghostscript can do antialiasing. It is memory intensive and if little memory is available the results are poor.

Hide (Show) Ghostscript messages

Error messages or general output messages from Ghostscript can be displayed by KGhostview. Select your preference here.

Hide (Show) toolbar

Toggle the toolbar on and off.

Hide (Show) statusbar

Toggle the statusbar on and off.

Configure key bindings ...

Lets you choose your own key bindings for the actions that KGhostview can perform.

Configure drivers ...

Lets you choose a few options about how the Ghostscript driver should work.

Save Options

Writes your set of preferences including the size of the KGhostview window and the location of the toolbar to a configuration file.

4.5 Help

About kghostview

Simply tells you the program name, version number and address of the author

Help for kghostview

Starts the KDE online help and loads in this handbook.
